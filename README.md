## 1. 实时字幕系统

**技术关键词**：实时字幕 | Live Caption | Subtitle | SRT | WebVTT | 视频通话字幕 | 直播字幕 | Speech-to-Text | STT

### 场景描述
为视频通话、在线课程、直播等场景提供实时字幕显示。

### 技术优势
- ✅ 本地识别延迟 < 1秒，体验流畅（**Apple Speech Framework**）
- ✅ 系统级捕获，不依赖应用 API（**Broadcast Upload Extension**）
- ✅ 画中画显示，不遮挡内容（**AVKit PiP**）
- ✅ 支持多语言切换（**Locale Identifier**）

### 实现方案

#### 视频通话字幕
```
用户场景：
1. 用户开启视频通话（微信/Zoom/Teams）
2. 启动录屏功能
3. 实时显示对话字幕
4. 支持中英文切换
```

#### 在线课程字幕
```
用户场景：
1. 学生观看在线课程
2. 自动生成实时字幕（Speech Framework 本地识别）
3. 支持字幕导出（SRT/WebVTT 格式）
4. 支持回放查看（时间戳同步）
```

### 商业价值
- **无障碍服务**：帮助听障用户
- **国际化工具**：多语言实时翻译
- **教育市场**：在线教育平台集成

### 目标客户
- 视频会议平台（Zoom、腾讯会议等）
- 在线教育平台（学而思、新东方等）
- 直播平台（抖音、快手等）
